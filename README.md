## Skills

- [fixing-cursor-remote-old-glibc](fixing-cursor-remote-old-glibc/SKILL.md) — Use when Cursor or VS Code remote-SSH fails to connect or the integrated terminal keeps dropping ("pty host connection dropped", "到 shell 进程的连接丢失", "ptyHost terminated unexpectedly with code 127", "Couldn't install Cursor Server ... system node is too old") on an old Linux server (CentOS 7, Kylin V10, or any glibc < 2.28), including arm64/aarch64 and x86_64.
